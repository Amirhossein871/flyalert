Installation
============

To install FlyAlert, run:

.. code-block:: bash

    pip install flyalert

Usage Example
=============

Here is a basic usage example:

.. code-block:: python

    from flyalert import FlyAlert

    config = {
        "title": "Warning",
        "message": "This is an alert",
        "icon": "warning",
        "ConfirmButton": True
    }

    alert = FlyAlert(config)
    alert.show()
